# Offline QR Code Generator

A simple, fast, and completely offline QR Code Generator.

![Offline QR Code Generator](./icon-512.png)

## Features

- **100% Offline**: No internet connection required after the first load. No data is sent to any server.
- **Privacy Focused**: All QR code generation happens locally in your browser.
- **PWA Support**: Install it on your device for an app-like experience.
- **Instant Download**: Download your QR codes as PNG images.

## Technology Stack

- HTML5
- CSS3 (Custom styles)
- JavaScript (Vanilla)
- [qrcode.js](https://davidshimjs.github.io/qrcodejs/) for QR generation
